    • 2.4 Configuración de la administración de switches

       Pasos para configurar un switch Cisco con CLI.

    • Paso 1: Utilice un emulador externo como Telnet o un PuTTY para iniciar sesión en el switch.

      • El símbolo del sistema inicial "Switch>" aparece en la pantalla.

      • Escriba "enable" a su lado y pulse "Enter".

      • Esto le llevará al modo "EXEC", también conocido como el modo Configuración global.

      • Vaya al modo configurar utilizando una terminal de configuración.

      • Ingrese los comandos de configuración, una por línea.

      Switch# configure terminal
      Switch(config)#


    • Paso 2: Suministre el nombre del host para que el switch funcione en un entorno de red particular

      Switch(config)#hostname switch
      Switch(config)#


    • Paso 3: Configure una contraseña de administración (habilite la contraseña secreta)

      Switch(config)#enable secret somestrongpass

      Nota: Esta contraseña se tendrá que dar antes de ingresar al modo configuración, una vez configurado.


    • Paso 4: Configure el gateway predeterminado

      Switch(config)# ip default-gateway IP-address
      Switch# show ip route


    • Paso 5: Configure la ruta estática

      Switch(config)# ip route dest_IP_address mask
      Switch# show running-config


    • Paso 6: Configure la descripción de la interfaz

      Switch(config)#interface fastethernet 0/1
      Switch(config-if)#description Development VLAN


    • Paso 7: Limpie la tabla de direcciones MAC

      switch#clear mac address-table


    • Paso 8: Ajuste el modo Dúplex

      Switch(config-if)#duplex full


    • Paso 9: Salga de la interfaz del modo configuración

      Switch(config-if)#exit
      Switch(config)#


    • Paso 10: Salga del modo configuración

      Switch(config)#exit
      Switch#


    • Paso 11: Copie la configuración en ejecución en la configuración de inicio con el siguiente comando

      Switch#write memory
      Building configuration... [OK]
      Switch#
